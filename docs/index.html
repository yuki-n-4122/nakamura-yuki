PROFILE

なかむら ゆきNAKAMURA Yuki

イラストレーター

1991年生まれ、岐阜県出身、東京都在住。
約3年間、WEBエンジニアとして仕事をし、物事をロジカルに考える力を身につけました。その後デザイン領域に専門を広げたいという想いで、約1年間ほどデザイン会社に勤務し、2021年からフリーのイラストレーターとして活動しています。きちんと感が求められる場面でもお使いいただけるような「ゆるすぎない可愛さ」を得意としております。

HOW TO ORDER

お問い合わせ

CONTACT記載のメールアドレスへお気軽にお問い合わせください。その際に、可能な範囲で構いませんので以下についてご明記いただけますと幸いです。

制作イラストの詳細／掲載媒体／ご予算／ご希望納期／ご希望フォーマット

お見積もり

メールなどで内容をすり合わせたのちにお見積もりやスケジュール感をお伝えします。ご承諾いただけましたら、正式なご注文となります。

イラスト制作

まずはラフ案をお送りします。そちらをもとに「もっとああしたいな」「こうだったらいいな」というご意見をお聞かせください。一緒により良いものを作れたら嬉しいです。OKをいただけましたら、本制作を進めます。

納品

完成品をメールまたはオンラインストレージで納品します。納品後、なかむらゆきのWEBサイトやSNSでの掲載許可を いただけますと幸いです。掲載NGの場合はその旨をご連絡ください。
